## Step1 Categorize tools

- Frontend

  - Framework
    - Vue.js

- Backend

  - Service
    - ASP.NET Core
  - Store
    - Redis
      Cache
    - SQL Server
      Data Store
  - Message Broker
    - Kafka

- Develop

  - Source Repo
    - Gitlab
  - Test Tools
    - Vitest
      frontend test framework, unit test, integration test
    - NUnit
      C# unit test
    - K6
      stress testing
  - Containerization
    - Docker

- Production
  - Deployment
    - K8s
  - Data collection & analyze
    - ELK (Elasticsearch, Logstash, Kibana)
      Event log, user behavior data collector
    - Prometheus
      system, applications, services data collector.
  - Data Visualization
    - Grafana
